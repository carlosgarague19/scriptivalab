## Scriptiva Lab | Landing Page

Este repositorio contiene el código fuente de la página de aterrizaje (landing page) de **Scriptiva Lab**, una empresa de ingeniería de software con propósito. 

El diseño destaca por ser minimalista, moderno y altamente optimizado, utilizando patrones geométricos y tipografía limpia para transmitir seguridad y vanguardia tecnológica.

---

### Tecnologías Utilizadas

*   **HTML5:** Estructura semántica y accesible.
*   **Tailwind CSS:** Framework de diseño (cargado vía CDN) para estilos responsivos y ágiles sin depender de hojas de estilo externas pesadas.
*   **Google Fonts:** Uso de la familia tipográfica **Inter** para garantizar la máxima legibilidad y estética moderna UI/UX.

---

### Características Principales

*   **Logotipo Dinámico en Código:** El isotipo (la "S" en grilla) y los patrones de fondo están construidos íntegramente con CSS Grid de Tailwind, eliminando la necesidad de imágenes (JPG/PNG/SVG) y maximizando la velocidad de carga.
*   **Diseño Responsivo:** Adaptabilidad perfecta en dispositivos móviles, tablets y pantallas de escritorio.
*   **Interactividad UI:** Efectos de sombras y elevación en tarjetas al pasar el cursor, junto con navegación de desplazamiento suave (*scroll-smooth*).
*   **Minimalismo Estructural:** Paleta de colores restringida a blanco, tonos de gris y negro para enfocar la atención en la propuesta de valor.

---

### Arquitectura de Despliegue (Deployment)

Este proyecto está configurado como un sitio web estático, optimizado para un alojamiento rápido y gratuito:

*   **Hosting:** Desplegado a través de **Render** (Static Site).
*   **DNS & Dominio:** Gestionado mediante **Hostinger**, utilizando un registro CNAME que enlaza el subdominio con los servidores de Render.
*   **CI/CD:** Integración continua directa con GitHub; cualquier *commit* a la rama principal actualiza la página en vivo instantáneamente.

---

### Instalación y Uso Local

Al ser un sitio estático puro sin dependencias de backend, no requiere de Node.js, npm ni configuraciones complejas para visualizarse.

1. Clona este repositorio en tu máquina local.
2. Abre el archivo `index.html` directamente en cualquier navegador web moderno (Chrome, Safari, Firefox, Edge).
3. Para editar, utiliza cualquier editor de código (como VS Code) y guarda los cambios para verlos reflejados al recargar la página.
